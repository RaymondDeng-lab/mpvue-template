# mpvue-template
* 这是一个基于`mpvue2.0` + `weapp(vant)`的项目模板。

* 使用了`mpvue-quickstart` 进行构建项目。[链接](https://github.com/F-loat/mpvue-quickstart)

## 插件
    1、`mpvue-entry` 分包
    2、`mpvue-router-patch`模拟路由跳转
